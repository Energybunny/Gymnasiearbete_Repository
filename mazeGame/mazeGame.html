How to play

This game is very simple: Move your character through the maze and reach the goal. The character is controlled using your keyboard with the following key bindings:

  • W: Move Up
  • A: Move Left
  • S: Move Down
  • D: Move Right
  • R: Reset the level
Currently, there are only four levels implemented and if you complete them all you start over.
